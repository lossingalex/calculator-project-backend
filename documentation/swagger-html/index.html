Calculate

Up 
post /calculate
Calculate te result of input formula (calculatePost)
Calculate te result of input formula. Any formula formatted following for mathjs can be used. (https://mathjs.org/examples/expressions.js.html)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

calculateRequest calculateRequest (optional)
Body Parameter — Any formula formatted following for mathjs can be used. (https://mathjs.org/examples/expressions.js.html)

Return type

calculateResponse

Example data

Content-Type: application/json
{
  "result" : "7"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Calculation result calculateResponse

400

Bad request - Missing parameters or invalid formula

Reporting

Up 
post /report/daily
Get all formula requests for a full day. (reportDailyPost)
Send back the calculation done during the full day from 00AM to midnight of the input date. Full day report is based on UTC time.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

reportRequest reportRequest (optional)
Body Parameter — Date for the report

Return type

historyCalculations

Example data

Content-Type: application/json
{
  "result" : [ {
    "result" : 10,
    "formula" : "30 / (2.3 + 0.7)",
    "id" : "0463424e-1925-49a6-84ee-4f0a81266867",
    "dateFormatted" : "2019-09-29T13:31:14.388Z",
    "type" : "CALCUL",
    "timestamp" : 1569763874388
  }, {
    "result" : 10,
    "formula" : "30 / (2.3 + 0.7)",
    "id" : "0463424e-1925-49a6-84ee-4f0a81266867",
    "dateFormatted" : "2019-09-29T13:31:14.388Z",
    "type" : "CALCUL",
    "timestamp" : 1569763874388
  } ]
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
